!!! Для запуска необходимы: Microsoft Visual Studio 2013 (желательно не Express!), .NET Framework 4.5

Там везде стандартные элементы управления, но обращу внимания на некоторые вещи.

1. ComboBox.ItemTemplate, ListBox.ItemTemplate

	Эта хрень задает стандартный шаблон для отображения элементов (внешний вид по сути)

2. Binding

	Это механизм WPF, позволяющий не писать ручками хуеву тучу кода, а привязывать значения полей объектов к элементам управления (TextBox, содержимое таблиц, списков и т.д.)

	Path - указывает имя поля, которое привязывается

	ElementName - указывает объект, который привязываем

	Mode - режим привязки (везде TwoWay стоит - изменять значение может как объект, так и элемент управления)

	UpdateSourceTrigger - задает поведение при изменении источник (в нашем случае стоит PropertyChanged - который вызывает изменение отображения при изменении свойств источника)

3. INotifyPropertyChanged

	Интерфейс, который позволяет отслеживать изменения объекта. Имеет в себе одно событие PropertyChanged. Нужен для Binding.

	Метод SetProperty<T> проверяет, изменено ли значение на новое, и, если изменено, то вызывает событие PropertyChanged.
